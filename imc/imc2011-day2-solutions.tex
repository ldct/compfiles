\section*{Problem 1}
Let $\tfrac{1}{2} < a_n < 1$ and $x_0 = a_0$, $x_{n+1} = \tfrac{a_{n+1} + x_n}{1 + a_{n+1} x_n}$. Find $\lim x_n$.

\subsection*{Solution}
We show by induction that $0 < 1 - x_n < \tfrac{1}{2^{n+1}}$, hence $x_n \to 1$.

Base $n = 0$: $\tfrac{1}{2} < x_0 < 1$. Step: from the recurrence,
\[
1 - x_{n+1} = \frac{1 - a_{n+1}}{1 + a_{n+1} x_n}(1 - x_n),
\]
and $0 < \tfrac{1 - a_{n+1}}{1 + a_{n+1} x_n} < \tfrac{1/2}{1} = \tfrac{1}{2}$, so $0 < 1 - x_{n+1} < \tfrac{1}{2}(1 - x_n) < \tfrac{1}{2^{n+2}}$.

\emph{Alt.} Using $\tanh(u+v) = \tfrac{\tanh u + \tanh v}{1 + \tanh u \tanh v}$, set $u_n = \operatorname{artanh} a_n$. Then $x_n = \tanh(u_0 + \cdots + u_n) \to 1$.

\section*{Problem 2}
Married-triples problem.

\subsection*{Solution}
(a) Split each of $M, F, E$ into halves. Draw edges only between different-indexed parts (so $M_1$--$F_2$, etc. but not $M_1$--$F_1$). Each vertex has degree $n/2$ to each other gender; but a 3-cycle must traverse an even number of 1$\to$2 swaps, while a 3-cycle has odd length --- so no 3-cycle exists.

(b) Assign arbitrary triples; let \emph{unhappiness} = number of non-liking pairs within triples. Show: if unhappiness $> 0$, we can swap an emale (say) to strictly reduce unhappiness.

Suppose $E_1$ dislikes someone in its triple. Need $i > 1$ with: $E_i$ likes both $F_1, M_1$, and $E_1$ likes both $F_i, M_i$. At most $n/4$ indices $i$ have $E_1$ disliking $F_i$, at most $n/4$ have $E_1$ disliking $M_i$; so at most $n/2$ bad indices (including $1$). Similarly at most $n/2$ indices where $M_1$ or $F_1$ dislikes $E_i$. Since each set has $\le n/2$ and both contain $1$, their union misses some index, giving a valid swap.

\emph{Alt (for b).} Use Hall's theorem. With $k \ge 3n/4$, any $A \subseteq M$ has neighbor set $B \subseteq F$ with $|B| \ge |A|$ (else an $f \in F \setminus B$ has $M$-degree $< n/4$, contradiction). Get a perfect matching $P$ between $M$ and $F$. Form bipartite graph $H$ on $P \cup E$: edge from $(m,f) \in P$ to $e \in E$ iff $(m, f, e)$ is a 3-cycle in $G$. Each $p \in P$ has $H$-degree $\ge 2k - n$. Verifying Hall's condition on $H$ yields the matching.

\section*{Problem 3}
Compute $\sum_{n \ge 1} \ln(1 + 1/n) \ln(1 + 1/(2n)) \ln(1 + 1/(2n+1))$.

\subsection*{Solution}
Let $f(n) = \ln\tfrac{n+1}{n}$; observe $f(2n) + f(2n+1) = f(n)$ and $f(n) \le 1/n$. Define
\[
g(n) = \sum_{k=n}^{2n-1} f^3(k) < n f^3(n) \le 1/n^2.
\]
Using $(a+b)^3 - a^3 - b^3 = 3(a+b)ab$,
\[
g(n) - g(n+1) = f^3(n) - f^3(2n) - f^3(2n+1) = 3 f(n) f(2n) f(2n+1).
\]
So
\[
\sum_{n=1}^N f(n) f(2n) f(2n+1) = \tfrac{1}{3}(g(1) - g(N+1)) \to \tfrac{1}{3} g(1) = \tfrac{1}{3} \ln^3 2.
\]
Answer: $\boxed{\tfrac{1}{3} \ln^3 2}$.

\section*{Problem 4}
Polynomial with $\tfrac{f(k) - f(m)}{k - m} \in \mathbb{Z}$ for $0 \le k < m \le n$. Show $(a - b) \mid f(a) - f(b)$ for all distinct $a, b \in \mathbb{Z}$.

\subsection*{Solution}
Let $L(k) = \operatorname{lcm}(1, \ldots, k)$, $h_k(x) = L(k)\binom{x}{k}$.

\textbf{Lemma.} $(a - b) \mid h_k(a) - h_k(b)$.

\emph{Proof.} From $\binom{a}{k} = \sum_{j=0}^k \binom{a-b}{j}\binom{b}{k-j}$,
\[
h_k(a) - h_k(b) = (a - b) \sum_{j=1}^k \frac{L(k)}{j} \binom{a-b-1}{j-1}\binom{b}{k-j},
\]
which is a multiple of $a - b$ since $L(k)/j \in \mathbb{Z}$.

Expand $f(x) = A_0 + A_1\binom{x}{1} + \cdots + A_n\binom{x}{n}$. By induction on $m$, show $L(m) \mid A_m$:
\[
\frac{f(m) - f(k)}{m - k} = \sum_{j=1}^{m-1} \frac{A_j}{L(j)} \cdot \frac{h_j(m) - h_j(k)}{m - k} + \frac{A_m}{m - k}.
\]
LHS and all terms but the last are integers, so $\tfrac{A_m}{m - k} \in \mathbb{Z}$ for all $0 \le k < m$; hence $L(m) \mid A_m$. By the lemma, $(a - b) \mid f(a) - f(b)$.

\section*{Problem 5}
Polygon reflection operations. Show $(f_1 \circ \cdots \circ f_{n-1})^n = \operatorname{id}$.

\subsection*{Solution}
The $f_i$ are rational maps; it suffices to verify identity on a neighborhood of the regular polygon.

Verify:
\begin{itemize}
  \item (i) $f_i \circ f_i = \operatorname{id}$.
  \item (ii) $f_i \circ f_j = f_j \circ f_i$ for $|i - j| \ge 2$.
  \item (iii) $(f_i \circ f_{i+1})^3 = \operatorname{id}$.
\end{itemize}
For (iii): $f_i, f_{i+1}$ permute the consecutive side lengths of quadrilateral $A_{i-1} A_i A_{i+1} A_{i+2}$; the sums of opposite angles are preserved. With fixed sides and these sums fixed, the quadrilateral is determined. After $(f_i f_{i+1})^3$, side lengths return to original order, so the quadrilateral is restored.

These are the Coxeter relations of $S_n$ with generators $\sigma_i = (i, i+1)$. Thus the group generated by the $f_i$ is a quotient of $S_n$. In $S_n$, $(\sigma_1 \sigma_2 \cdots \sigma_{n-1})^n = (1\,2\,\cdots\,n)^n = \operatorname{id}$, hence $(f_1 \circ \cdots \circ f_{n-1})^n = \operatorname{id}$.

\section*{Problem 1}
Find all continuous functions $f : \mathbb{R} \to \mathbb{R}$ such that $f(x) - f(y)$ is rational whenever $x - y$ is rational.

\subsection*{Solution}
Answer: $f(x) = ax + b$ where $a \in \mathbb{Q}$, $b \in \mathbb{R}$.

For each rational $q$, $g_q(x) = f(x + q) - f(x)$ is continuous and rational-valued, hence constant. Let $a = f(1) - f(0)$, $b = f(0)$. For $r = f(1/n) - f(0)$: telescoping gives $f(k/n) - f(0) = kr$. For $k = n$: $a = nr$, so $r = a/n$, hence $f(k/n) = a \cdot k/n + b$ for all integers $k, n > 0$. By density and continuity, $f(x) = ax + b$.

\section*{Problem 2}
Let $V$ be the real vector space of all real polynomials in one variable, $P : V \to \mathbb{R}$ linear. Suppose $P(fg) = 0$ implies $P(f) = 0$ or $P(g) = 0$. Prove there exist $x_0, c \in \mathbb{R}$ such that $P(f) = c f(x_0)$ for all $f$.

\subsection*{Solution}
WLOG $P \ne 0$. If $P(f) \ne 0$, then $P(f^2) \ne 0$, so $P(f^2) = a P(f)$ for some $a \ne 0$. From $P(f^2 - af) = P(f(f-a)) = 0$ we get $P(a) \ne 0$. Rescale so $P(1) = 1$. Replace $P$ by $\tilde P(g(X)) = P(g(X+b))$ where $b = -P(X)$, so $P(X) = 0$.

Inductively show $P(X^k) = 0$ for $k \ge 1$. Assume true for $k < n$, and set $e = -P(X^n)$. From $P((X+e)(X+1)^{n-1}) = P(X^n + e) = 0$ and $P(X + 1) = 1 \ne 0$, we get $P(X + e) = 0$, hence $e = 0$. So $P(X^n) = 0$.

Combined with $P(1) = 1$: $P(f) = f(0)$, the evaluation map.

\section*{Problem 3}
Let $p \in \mathbb{Z}[x]$, $a_1 < \cdots < a_k$ integers. (a) Prove $\exists a \in \mathbb{Z}$ with $p(a_i) \mid p(a)$ for all $i$. (b) Does there exist $a$ with $\prod_i p(a_i) \mid p(a)$?

\subsection*{Solution}
WLOG all $p(a_i)$ nonzero and distinct. Choose coprime $s, t$ with $s \mid p(a_1)$, $t \mid p(a_2)$, $st = \operatorname{lcm}(p(a_1), p(a_2))$. Find $m, n$ with $a_1 + sn = a_2 + tm = b_2$. Then $s \mid p(b_2) - p(a_1)$ and $t \mid p(b_2) - p(a_2)$, so $st \mid p(b_2)$. Induct to get $a = b_k$ with all $p(a_i) \mid p(b_k)$.

(b) No: $p(x) = 2x^2 + 2$ has $p(0) = 2$, $p(1) = 4$; no $p(a)$ divisible by $8$.

\section*{Problem 4}
A triple $(a_1, a_2, a_3)$ of nonnegative reals is \emph{better} than $(b_1, b_2, b_3)$ if at least two of $a_i > b_i$ hold. A triple $(x,y,z)$ is \emph{special} if nonnegative and $x + y + z = 1$. Find all $n$ for which there is a set $S$ of $n$ special triples such that every special triple has a better triple in $S$.

\subsection*{Solution}
Answer: $n \ge 4$.

Example with $n = 4$: $\{(0, \tfrac{8}{15}, \tfrac{7}{15}), (\tfrac{2}{5}, 0, \tfrac{3}{5}), (\tfrac{3}{5}, \tfrac{2}{5}, 0), (\tfrac{2}{15}, \tfrac{11}{15}, \tfrac{2}{15})\}$ works by case analysis.

Any 3 triples fail: given $S = \{T_1, T_2, T_3\}$, set $a = \min_i x_i$, etc. If $a + b + c \ge 1$ then all three triples are equal. Else by rescaling and transposition, WLOG $a = b = c = 0$ with $x_1 = y_2 = z_3 = 0$; one of $(x_2, 1 - x_2, 0)$, $(z_1, 0, 1 - z_1)$, $(0, y_3, 1 - y_3)$ works as an uncovered special triple. (Pair sums: $x_2 + y_1$, $z_1 + x_3$, $y_3 + z_2$ total $\le 3$ after normalization; some pair sums to $\le 1$.)

\section*{Problem 5}
Does there exist a finite group $G$ with a normal subgroup $H$ such that $|\operatorname{Aut} H| > |\operatorname{Aut} G|$?

\subsection*{Solution}
\textbf{Yes.} Let $H = \mathbb{F}_2^3$; then $|\operatorname{Aut} H| = |GL_3(\mathbb{F}_2)| = 168$.

Let $\phi : H \to H$ be the cyclic shift $(x_1, x_2, x_3) \mapsto (x_2, x_3, x_1)$, order 3. Let $T = \langle a : a^3 = e\rangle$ act on $H$ via $\phi$, and $G = H \rtimes T$, $|G| = 24$. So $G = \{ba^i : b \in H, i \in \mathbb{Z}/3\}$, $ab = \phi(b)a$.

Count orders: 1 identity; 7 nontrivial elements of $H$ have order 2. For $g = ba$: $g^3 = \psi(b)$ where $\psi(b) = b + \phi(b) + \phi^2(b) = (x_1+x_2+x_3)(1,1,1)$. So $\psi(b) = 0$ iff $x_1 + x_2 + x_3 = 0$ (4 such $b$). Similarly for $ba^2$. So $G$ has $8$ elements of order 3 ($b \in \ker \psi$) and $8$ elements of order 6 ($b \notin \ker \psi$).

$G$ is generated by any $b_0 \notin \ker\psi$ and $a$. An automorphism sends $b_0$ to an element of order 6 (8 choices) and $a$ to an element of order 3 (8 choices, but not all work). So $|\operatorname{Aut} G| \le 56 < 168$.

\section*{Problem 6}
Let $\sigma = (i_1, \ldots, i_n)$ be a permutation of $(1, \ldots, n)$. Let $D(\sigma) = \sum |i_k - k|$ and $Q(n, d) = \#\{\sigma : D(\sigma) = d\}$. Prove $Q(n, d)$ is even for $d \ge 2n$.

\subsection*{Solution}
Consider the determinant of the matrix $[x^{|i - j|}]_{ij}$:
\[
\Delta(x) = \sum_{\sigma \in S_n} (-1)^{\operatorname{inv}(\sigma)} x^{D(\sigma)}.
\]
So $Q(n, d)$ equals the coefficient of $x^d$ in $\Delta(x) \pmod 2$.

Evaluate $\Delta$: subtract $x \cdot$(row $k-1$) from row $k$ successively. This zeros out below the diagonal, leaving upper-triangular with diagonal entries $1, 1 - x^2, 1 - x^2, \ldots, 1 - x^2$:
\[
\Delta(x) = (1 - x^2)^{n-1}.
\]
This has degree $2(n-1) = 2n - 2$; coefficient of $x^d = 0$ for $d \ge 2n$. So $Q(n, d)$ is even.

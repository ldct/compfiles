\section*{Problem 1}
$a_1 = 1$, $a_{n+1} > \tfrac{3}{2} a_n$. (a) Prove $a_n / (3/2)^{n-1}$ has a finite limit or tends to $\infty$. (b) For every $\alpha > 1$, construct such a sequence with limit $\alpha$.

\subsection*{Solution}
(a) $b_n = a_n / (3/2)^{n-1}$; $a_{n+1} > \tfrac{3}{2} a_n$ iff $b_{n+1} > b_n$. Increasing sequences converge or diverge to $+\infty$.

(b) Pick any increasing sequence $1 = b_1 < b_2 < \cdots \to \alpha$; set $a_n = (3/2)^{n-1} b_n$.

\section*{Problem 2}
Let $a_1, \ldots, a_{51}$ be nonzero elements of a field. Replace each by the sum of the other 50; if the result is a permutation of the original, what can the characteristic be?

\subsection*{Solution}
Let $S = \sum a_i$. Then $b_i = S - a_i$, and $\sum b_i = 50 S$. Since $(b_i)$ is a permutation of $(a_i)$: $50 S = S$, so $49 S = 0$. If char $\ne 7$: $S = 0$, so $b_i = -a_i$; $(a_i)$ is a permutation of $(-a_i)$, pairing $a_i$ with $-a_{\varphi(i)}$. If char $\ne 2$, this partitions the $51$ elements into pairs, impossible (odd). So char $\in \{2, 7\}$.

Both achievable: char $= 7$, $a_i = 1$; char $= 2$, any $a_i$ with $S = 0$.

\section*{Problem 3}
$A$ real $n \times n$ with $3 A^3 = A^2 + A + I$. Show $A^k$ converges to an idempotent.

\subsection*{Solution}
Minimal polynomial of $A$ divides $3 x^3 - x^2 - x - 1 = (x - 1)(3x^2 + 2x + 1)$. Roots: $1$ and $\tfrac{-1 \pm i \sqrt 2}{3}$, both of modulus $1/\sqrt 3 < 1$. All three distinct, so $A$ is diagonalizable: $A = C^{-1} D C$. $D^k$ has diagonal entries tending to $0$ or $1$; limit $M$ is diagonal $0/1$, hence idempotent. $A^k \to C^{-1} M C$, idempotent.

\section*{Problem 4}
Find all pairs $(a, b)$ of positive integers such that $\mathbb{Z}^+$ can be split as $A \sqcup B$ with $a A = b B$.

\subsection*{Solution}
WLOG $\gcd(a, b) = 1$ (divide out). From $1 \in A$ or $B$, one of $a, b$ divides the other. So WLOG $a = 1$, $b = n \ge 2$.

For $n \ge 2$: define $f(k) = $ largest $j$ with $n^j \mid k$. Let $A = \{k : f(k) \text{ odd}\}$, $B = \{k : f(k) \text{ even}\}$. Then $k \in A \iff nk \in B$ since $f(nk) = f(k) + 1$. So $A = n B$.

Answer: all $(a, b)$ with $\gcd(a, b) = d$ and $(a/d, b/d) = (1, n)$ or $(n, 1)$ for $n \ge 2$.

\section*{Problem 5}
$g : [0,1] \to \mathbb{R}$ continuous; $f_0 = g$, $f_{n+1}(x) = \tfrac{1}{x}\int_0^x f_n$. Find $\lim f_n(x)$.

\subsection*{Solution}
Answer: $g(0)$ for all $x \in (0, 1]$.

\emph{Proof for nondecreasing $g$:} by induction, each $f_n$ is nondecreasing and $g = f_0 \ge f_1 \ge \cdots \ge g(0)$. So $h(x) = \lim f_n(x)$ exists. Suppose $h(1) > g(0)$; pick $\delta$ with $g(\delta) < h(1)$. Then
\[
f_{n+1}(1) \le \int_0^\delta g + \int_\delta^1 f_n \le \delta g(\delta) + (1 - \delta) f_n(1),
\]
giving $f_n(1) - f_{n+1}(1) \ge \delta(h(1) - g(\delta)) > 0$, so $f_n(1) \to -\infty$. Contradiction.

\emph{General $g$:} Let $M(x) = \sup_{[0,x]} g$, $m(x) = \inf_{[0,x]} g$ (nondecreasing and nonincreasing respectively, both continuous, $M(0) = m(0) = g(0)$). Apply the operator to $M, m$ to get $M_n, m_n$; by induction $m_n \le f_n \le M_n$. Squeeze.

\emph{Alt. explicit formula:} $f_{n+1}(x) = \tfrac{1}{x} \int_0^x g(t) \tfrac{(\log(x/t))^n}{n!} dt$ (by Fubini, after computing iterated integral).

\section*{Problem 6}
$f(z) = \sum a_k z^k$ real coefficients, all roots with $\operatorname{Re} < 0$. Prove $a_k a_{k+3} < a_{k+1} a_{k+2}$ for $0 \le k \le n - 3$.

\subsection*{Solution}
Factor $f$ over $\mathbb{R}$: linear factors $(k_i z + l_i)$ with $k_i, l_i > 0$ (root is $-l_i/k_i$, negative), quadratic $(p_j z^2 + q_j z + r_j)$ with $p_j, q_j, r_j > 0$ (roots have negative real part). WLOG all coefficients of $f$ positive.

Extend $a_k = 0$ for $k < 0$ or $k > n$. Induct on $n$. $n \le 2$: trivial ($a_k a_{k+3} = 0$).

$n \ge 3$: factor $f = (z^2 + pz + q) g$ with $p, q > 0$, $g = \sum b_k z^k$. Then $a_k = q b_k + p b_{k-1} + b_{k-2}$ (convention $b_k = 0$ outside range). By induction $b_{k+1} b_{k+2} \le b_k b_{k+3}$ for all $k$.

Compute
\begin{align*}
a_{k+1} a_{k+2} - a_k a_{k+3} &= (b_{k-1} b_k - b_{k-2} b_{k+1}) + p(b_k^2 - b_{k-2} b_{k+2}) + q(b_{k-1} b_{k+2} - b_{k-2} b_{k+3}) \\
&\quad + p^2(b_k b_{k+1} - b_{k-1} b_{k+2}) + q^2(b_{k+1} b_{k+2} - b_k b_{k+3}) + pq(b_{k+1}^2 - b_{k-1} b_{k+3}).
\end{align*}
Each term is $\ge 0$ by induction hypothesis; the $p^2$-term is $> 0$ (both coefficients in the valid range, $0 \le k \le n - 3$). E.g., to check $b_k^2 \ge b_{k-2} b_{k+2}$:
\[
b_{k-1}(b_k^2 - b_{k-2} b_{k+2}) = b_{k-2}(b_k b_{k+1} - b_{k-1} b_{k+2}) + b_k(b_{k-1} b_k - b_{k-2} b_{k+1}) \ge 0.
\]
Similar manipulations for the other cross terms.

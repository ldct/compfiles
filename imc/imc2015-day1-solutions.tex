\section*{Problem 1}
\subsection*{Solution}
Multiply the given equation by $(A+B)$:
\[ I = (A+B)(A^{-1} + B^{-1}) = I + AB^{-1} + BA^{-1} + I, \]
so $AB^{-1} + BA^{-1} + I = 0$. Let $X = AB^{-1}$; then $X + X^{-1} + I = 0$. Multiplying by $(X-I)X$: $(X-I)(X^2+X+I) = X^3 - I = 0$. So $X^3 = I$, hence $(\det X)^3 = 1$, and over $\mathbb{R}$, $\det X = 1$. Thus $\det A = \det X \cdot \det B = \det B$.

Over $\mathbb{C}$: take $A = I$ and $B = \omega I$ where $\omega = e^{2\pi i/3}$ (for $n$ not divisible by $3$); then $1 + \omega + \omega^2 = 0$ and the condition holds, but $\det B \ne 1$.

\section*{Problem 2}
\subsection*{Solution}
If $2^{r-1} \le k < 2^r$ then $k + f(k) = 2^r - 1$. Assume $2^{s-1} - 1 \le n \le 2^s - 1$. Then
\[ \frac{n(n+1)}{2} + \sum_{k=1}^n f(k) = \sum_{k=1}^n (k+f(k)) = \sum_{r=1}^{s-1} 2^{r-1}(2^r - 1) + (n - 2^{s-1} + 1)(2^s - 1) = (2^s - 1) n - \tfrac{1}{3} 4^s + 2^s - \tfrac{2}{3}. \]
After simplification,
\[ \frac{n^2}{4} - \sum_{k=1}^n f(k) = \frac{3}{4}\Big(n - \frac{2^{s+1} - 2}{3}\Big)\Big(n - \frac{2^{s+1} - 4}{3}\Big). \]
One factor is an integer (depending on parity of $s$) and the difference between factors is $< 1$, so either zero or same-sign. Equality iff $n = \frac{2^{s+1} - 2}{3}$ ($s$ even) or $n = \frac{2^{s+1} - 4}{3}$ ($s$ odd).

\section*{Problem 3}
\subsection*{Solution}
Characteristic polynomial $x^2 - \tfrac{5}{2} x + 1 = 0$ has roots $2, 1/2$. With initial data, $F(n) = 2^n - 2^{-n}$. Then
\[ \frac{1}{F(2^n)} = \frac{2^{2^n}}{(2^{2^n})^2 - 1} = \frac{1}{2^{2^n} - 1} - \frac{1}{2^{2^{n+1}} - 1}, \]
which telescopes to $\sum_{n=0}^\infty \frac{1}{F(2^n)} = \frac{1}{2^{2^0} - 1} = 1$. Rational.

\section*{Problem 4}
\subsection*{Solution}
Suppose such integers exist. Then $z_1 = 1 + 16 i$ and $z_2 = \prod_{k=1}^{15}(1+ki)^{m_k}$ have the same argument, so $R = z_2/z_1 \in \mathbb{R}^*$, and $R$ is a nonzero integer (since $\mathrm{Re}(z_1) = 1$, $\mathrm{Re}(z_2) \in \mathbb{Z}$). Taking absolute values squared:
\[ (1 + 16^2) R^2 = \prod_{k=1}^{15}(1+k^2)^{m_k}. \]
But $p = 257$ is prime (Fermat prime); $p$ appears with odd exponent on LHS, but on RHS all prime factors are $< p$ (since $1 + k^2 < 257$ for $k \le 15$). Contradiction.

\section*{Problem 5}
\subsection*{Solution}
Let $\mathbf{v}_i = \overrightarrow{BA_i}$. Then $\angle A_i B A_j > 90^\circ \iff \mathbf{v}_i \cdot \mathbf{v}_j < 0$. Since $B$ is interior, $\sum w_i \mathbf{v}_i = 0$ with $w_i > 0$.

Form graph on $\{1, \dots, n+1\}$: edge $\{i,j\}$ iff $\mathbf{v}_i \cdot \mathbf{v}_j < 0$. A connected graph on $n+1$ vertices has $\ge n$ edges, so it suffices to show this graph is connected.

Else partition $V, W$. Then
\[ 0 = \Big(\sum_i w_i \mathbf{v}_i\Big)^2 = \Big(\sum_{i \in V} w_i \mathbf{v}_i\Big)^2 + \Big(\sum_{i \in W} w_i \mathbf{v}_i\Big)^2 + 2\sum_{i \in V, j \in W} w_i w_j (\mathbf{v}_i \cdot \mathbf{v}_j), \]
all terms $\ge 0$. But first two are nonzero (else $\mathbf{v}_i$'s dependent, contradicting non-hyperplanarity), contradiction.

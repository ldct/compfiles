\section*{Problem 1}
\subsection*{Solution}
Answer: $4^{n-1} + 2^{n-1}$.

Using generating functions: consider
\[ \frac{(a+b+c+d)^n + (-a-b+c+d)^n + (-a+b+c+d)^n + (a-b+c+d)^n}{4}. \]
The coefficient of $a^A b^B c^C d^D$ equals $\frac{(1 + (-1)^{A+B} + (-1)^A + (-1)^B)}{4} = \frac{(1+(-1)^A)(1+(-1)^B)}{4}$, which is $1$ if $A, B$ are both even, else $0$.

Substituting $a = b = c = d = 1$: $(4^n + 0 + 2^n + 2^n)/4 = 4^{n-1} + 2^{n-1}$.

\section*{Problem 2}
\subsection*{Solution}
Let $X = AB - BA$. The trace is cyclic, so
\[ \operatorname{tr}(X^2) = \operatorname{tr}(ABAB - ABBA - BAAB + BABA) = 2(\operatorname{tr}(ABAB) - \operatorname{tr}(A^2 B^2)). \]
We need $\operatorname{tr}(X^2) = n(n-1)$. Since $X + I$ has rank 1, its eigenvalues are $(n, 0, \ldots, 0)$ or we write $X + I = vw^T$. Then $\operatorname{tr}(X) = 0$ gives $\operatorname{tr}(vw^T) = w^T v = n$. So
\[ X^2 = (vw^T - I)^2 = I - 2vw^T + vw^T v w^T = I + (w^T v - 2) vw^T, \]
$\operatorname{tr}(X^2) = n + (n-2) n = n(n-1)$.

\section*{Problem 3}
\subsection*{Solution}
Take an inclusion-maximal set $\{p_1, \ldots, p_m\} \subseteq \partial K$ such that the sets $K_i = p_i + \tfrac{\varepsilon}{2} K$ have pairwise disjoint interiors. Let $L = \operatorname{conv}\{p_1, \ldots, p_m\}$.

\textbf{Counting.} Each $K_i$ lies in $(1 + \varepsilon/2) K \setminus (1 - \varepsilon/2) K$ (using symmetry and $p_i \in \partial K$). Volume: $m (\varepsilon/2)^d \operatorname{vol}(K) \leq ((1+\varepsilon/2)^d - (1-\varepsilon/2)^d) \operatorname{vol}(K) \leq (3/2)^d \varepsilon \operatorname{vol}(K)$, so $m \leq 3^d \varepsilon^{1-d}$.

\textbf{Inclusion $(1-\varepsilon)K \subseteq L$.} Suppose $p \in (1-\varepsilon)K \setminus L$. Separate by functional $\ell$ with $\ell(p) > \max_i \ell(p_i)$. Choose $x \in K$ maximizing $\ell(x) = a$. By maximality of $\{p_i\}$, $x + \tfrac{\varepsilon}{2} K$ meets some $K_i$. Then $\ell(p_i) + \tfrac{\varepsilon}{2} a \geq \ell(x) - \tfrac{\varepsilon}{2} a$, so $\ell(p_i) \geq a(1-\varepsilon) \geq \ell(p)$, contradiction.

\section*{Problem 4}
\subsection*{Solution}
Let $h(z) = p(1 - \bar z) - p(z)$. For $t \in \mathbb{R}$: $h(it) = p(1 + it) - p(it) = t^{100}$ (since $p(x+1) - p(x) = x^{100}$ for real $x$, extended by polynomial identity), and $h(1/2 + it)$ has real part $0$ by symmetry.

If $p = \sum c_j z^j$, deg $p = n$:
\[ h(a + it) = (1 - 2a)(n c_n i^{n-1} t^{n-1} + Q(t, a)) \]
with $Q$ of degree $\leq n-2$ in $t$. Setting $a = 0$ and matching $t^{100}$: $n = 101$, $c_{101} = 1/101$.

For large $|t|$ and $a \in [0, 1/2)$, $\operatorname{Re}(h(a + it))$ is dominated by positive terms. By maximum principle for harmonic $\operatorname{Re} h$ on the rectangle $[0, 1/2] \times [-N, N]$ (with $\operatorname{Re} h \geq 0$ on the boundary for large $N$), $\operatorname{Re} h \geq 0$ throughout, in particular on $[0, 1/2]$: $p(1-t) - p(t) \geq 0$.
